# Implementación HU2 – Agendamiento de cita médica

Esta historia de usuario permite que el paciente seleccione:

- Especialidad
- Médico
- Fecha
- Hora disponible

## Flujo básico

1. El paciente accede al módulo **"Agendar cita"** desde el menú principal.
2. El sistema carga la lista de especialidades disponibles.
3. El paciente selecciona la especialidad y el médico.
4. El sistema consulta la disponibilidad del médico y muestra los horarios libres.
5. El paciente elige fecha y hora y confirma la cita.
6. El sistema guarda la cita en la base de datos y actualiza la agenda del médico.
